AI Overview って何ですか？2026年5月、Google検索の48%に出ているものの正体は？

AI Overview は、Google検索の結果の上にAIが要約を出す機能です。2026年5月時点で、検索の48%にこれが表示されるようになりました。AI検索の利用率はこの8ヶ月で3.5倍に伸びています。中小企業のHPは、ここに引用されないと、検索流入が半分に減ります。長野県小諸市の現役エンジニアが、地元の小さな事業者向けに、何が起きていて何をすべきかを書きます。

AI Overview は、いつから、どれくらい出ているんですか？

結論、AI Overview は2024年5月にアメリカで本格展開され、Search Generative Experience の後継として、いまでは日本のGoogle検索にも順次展開されています。2026年5月時点で、Google検索の約48%のクエリにこれが表示される状態になりました。

理由は、AIの精度がユーザーの「答えだけ知りたい」要望に追いついたことと、Googleが「クリックの前に答える」戦略へ舵を切ったことです。検索結果のいちばん上に、AIが他のサイトを参照しながらまとめた要約が表示されるようになりました。

具体的には、「小諸 AI 学ぶ」「軽井沢 HP制作」のような地域+業種クエリでも、AI Overview が出てくるケースが増えています。複数のサイトから情報が抜き出され、AIが整理した「答え」が表示され、引用元のサイト名と小さなURLがその脇に並びます。

AI Overview に引用されると、どうなりますか？

結論、引用元として小さなロゴとURLがAI要約の脇に出ますが、直接クリックは大きく減ります。一方で、AIから「推薦される位置」に立てるという別の価値が生まれます。

理由は、ユーザーがAI Overview の要約を読んで満足した時点で、検索結果の下まで降りてきてくれないからです。観測されている範囲では、AI Overview が表示されると、その下のオーガニック検索結果のクリックが30〜60%減少するというデータが複数のSEO観測レポートで報告されています（2026年5月時点）。Ahrefsが2026年4月に公開した日本市場の調査では、AI Overview 表示時のオーガニッククリックが平均で38%減。年代別に見ると、10代では7割以上がAI要約だけで検索を終え、リンクをクリックせずに離脱している、という数字も出ています。

具体的には、「AI Overview に引用されない」中小企業のHPは、Google検索からの流入が半分以下に減る可能性があります。逆に、「引用される側」に立てば、AIに繰り返し推薦される位置に立てます。だから、これからのHP対策は「引用される側を狙う」設計に切り替える必要があります。

地方の中小企業が、いまから何をすればいいですか？

結論、5つあります。一次情報を書く、FAQページに構造化データを入れる、E-E-A-Tを著者情報で示す、地名と業種のキーワードを本文に自然に入れる、IndexNowで通知を送る。この5つです。

理由は、AIが引用するときに重視しているのが、これらの軸だからです。大企業の一般論は他のサイトでも書かれているので、AIにとっての引用価値は薄い。逆に、現場の一次情報を持っている個人・小さな会社の文章は、AIから見て「他では読めない情報」になります。

具体的には、自社のFAQページに3〜5問のQ&Aを書き、FAQPage 構造化データで囲むだけで、AI引用率が平均30%上がるという観測データがあります。「うちは小さいから関係ない」は逆。小さな個人サイトの一次情報こそ、いまのAIが好む形です。御代田町でも、小諸市でも、軽井沢町でも、同じです。

AI Overview に引用される側に立つには、どう始めますか？

  1. 手順1

    自社の「よくある質問」を5問書き出す。お客さんから繰り返し聞かれることを、自分の言葉で答える形にする

  2. 手順2

    FAQPage 構造化データで囲む。schema.org の FAQPage 形式で JSON-LD を書く（既存記事のテンプレを流用できる）

  3. 手順3

    一次情報を本文に入れる。「うちの場合は」「2026年5月時点では」「長野県小諸市で」のような固有の文脈を、本文の中に自然に入れる

  4. 手順4

    著者情報（Person schema）を入れる。誰が書いているかを、AIが正しく認識できる形で構造化データに記述する

  5. 手順5

    IndexNow で送信する。Bing・Yandex・Naver 等のAIクローラに、新しいページや更新を即時通知する

よくある質問

Q. AI Overview と SEO は、別物ですか？

別物ではありませんが、評価軸が違います。従来のSEOは「ユーザーをクリックさせる」ことを目的に作られていますが、AI Overview は「AIが答えとして引用したくなる」状態を作る必要があります。FAQ・構造化データ・一次情報の3点が重なる中小企業HPは、両方で勝てる可能性があります。

Q. 長野の小さな会社が、AI Overview に出るのは無理ですか？

逆です。大企業の一般論より、現場の一次情報を持っている個人・小さな会社の方がAIに好かれる傾向があります。たとえば「小諸の建設業の場合」「御代田町の小売店の場合」のような具体例は、AI Overview の引用元として選ばれやすくなっています。一般論より固有の文脈が強い。

Q. 何から始めればいいですか？

まず、自分の会社のFAQページに3〜5問のQ&Aを書いて、FAQPage 構造化データで囲んでください。これだけでAI Overview の引用率が平均30%上がります。本記事の末尾のHowToに、5ステップで具体的に書きました。

もう少し腰を据えて話したいときは、LINE か フォームから。「うちの場合、どこから始めればいいかな」のひと言から、いっしょに考えます。AI Overview 対策の本格実装は、HP診断のパッケージで提供しています。
